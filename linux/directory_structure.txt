/ -> root directory
/bin -> binaries
/root -> root user's home directory
/tmp -> temp files, deleted on restart
/home -> houses user directories
/home/username -> specific users directory
/etc -> ectetera file. where config files sit
/etc/fstab -> system config file that contains all available disks, disk partitions and their options
/dev -> devices, where HDD's are found
/mnt -> For directories/devices that are mounted manually
/media -> Drives that are automatically mounted by your system
